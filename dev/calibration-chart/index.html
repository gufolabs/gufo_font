    Calibration Chart¶

    This calibration chart is used for debugging and testing purposes.

    Bounding Box¶

    X sign must be surrounded by red box (without gap):

    Inline Text¶

    Logo must be aligned with following text on baseline:

    Gufo Labs

    Larger text:

    Gufo Labs

    Horizontal Gap¶

    Four X signs must touch each other.

    Canvas¶

    Square Icon¶

    You must see turquose square with centered darker square, red cross and the router perfectly aligned on center. It must fill full width and height of the gray square.

    Horizontal Icon¶

    You must see turquose square with centered darker square, red cross and the router perfectly aligned on center. It must fill full width of the gray square and to be vertically centered.

    Vertical Icon¶

    You must see turquose square with centered darker square, red cross and the router perfectly aligned on center. It must fill full height of the gray square and to be horizontally centered.
